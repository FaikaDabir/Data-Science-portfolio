# 📊 Advanced Data Science & Machine Learning Portfolio

Welcome to my core data science portfolio. This repository serves as a centralized hub hosting production-ready machine learning pipelines, multi-model algorithmic benchmarks, heuristic AI models, and advanced business intelligence frameworks. 

---

## 🛠️ Portfolio Overview & Core Projects

### 1. 🚢 [End-to-End Machine Learning Pipeline](Titanic_survival_prediction_ML_pipeline.ipynb)
* **Objective:** Built an automated, highly modular feature-engineering and predictive pipeline using the Titanic survival dataset.
* **Tech Stack:** Python, Scikit-Learn (Pipelines, ColumnTransformer), Pandas, NumPy.
* **Key Achievements:** Standardized numeric features via `StandardScaler` and encoded categorical strings using `OneHotEncoder`. Isolated structural leakage parameters and achieved clear baseline predictive accuracy.

### 2. 🍷 [Multi-Model Classification & Algorithmic Benchmarking](Wine_Quality_classification.ipynb)
* **Objective:** Built a multi-model evaluation framework to predict wine quality traits.
* **Tech Stack:** Python, Scikit-Learn, Pandas, NumPy.
* **Key Achievements:** Benchmarked Logistic Regression, KNN, Gaussian Naive Bayes, and Random Forest. Achieved a peak classification accuracy of **69.87%** utilizing an ensemble Random Forest architecture. Evaluated class imbalances using precision/recall auditing matrices.

### 3. 📈 [Advanced Business Intelligence & Product Performance Dashboard](ECommerce_sales_Advanced_EDA_Dashborad.ipynb)
* **Objective:** Engineered a comprehensive 2x2 multi-plot analytics dashboard to decode E-Commerce sales dynamics.
* **Tech Stack:** Python, Matplotlib, Seaborn, Pandas.
* **Key Achievements:** Designed standardized visualization grid matrices mapping skewness distributions, pricing outlier quartiles (box plots), categorical market breakdowns, and numeric variable correlation metrics.

### 4. 🧠 [End-to-End NLP Sentiment Analysis Engine](Product_review_Sentiment_analysis_NLP.ipynb)
* **Objective:** Developed a production-ready natural language processing text classifier.
* **Tech Stack:** Python, NLTK, Scikit-Learn, Pickle.
* **Key Achievements:** Built text processing workflows handling normalization, tokenization, stopword filtering, and WordNet lemmatization. Structured numeric text weights using `TfidfVectorizer` and serialized model binaries via `pickle` for instant deployment.

### 🧗 [Heuristic AI & Search Optimization](Heauristic_function.ipynb)
* **Objective:** Implemented mathematical heuristic functions and state-space traversal models using local search optimizations.
* **Tech Stack:** Python, NumPy.
* **Key Achievements:** Engineered a functional **Hill Climbing optimization algorithm** to iteratively maximize objective search criteria, evaluating state adjustments against local maxima constraints.
